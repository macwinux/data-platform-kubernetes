# Flink Operator

## Commands

### Install

`dp flinkop install`

Options

- `--version` or `-v`: version of the flink operator.

### Delete

`dp flinkop delete`


### Deploy a test

`dp flinkop deploy-test -n test test-deploy.yaml`

Options

- `--namespace` or `-n`: namespace where to deploy

Arguments
- Yaml to deploy with flink deployment.
