# MinIO

## Commands

### Install

`dp miniop install`

### Delete

`dp miniop delete`


### Get JWT

`dp miniop get-jwt` 


### Create Tenant

`dp miniop create-tenant -n minio-tenant`

Options

- `--namespace` or `-n` Namespace where MinIO tenant will be deployed
 
 ### Delete Tenant

`dp miniop delete-tenant -n minio-tenant`

Options

- `--namespace` or `-n` Namespace where MinIO tenant will be deleted
