# Redpanda

## Commands

### Install

`dp redpanda install`

Options

- `--version` or `-v`: Redpanda helm chart version
- `--tls` or `-t`: Set TLS configuration in Redpanda

### Delete

`dp redpanda delete`


### Deploy a test

`dp redpanda`

Options

- namespace: namespace where to deploy

Arguments
- Yaml to deploy with flink deployment.
